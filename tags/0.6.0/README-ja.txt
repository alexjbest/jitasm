jitasm
x86/x64 JIT Assembler Library
http://code.google.com/p/jitasm/

概要
====
jitasmはx86/x64のコードを動的に生成するためのC++ライブラリです。
インラインアセンブラのような直感的な記述で動的なコード生成・実行を行うことができます。

ライセンス
==========
jitasmはオープンソースソフトウェアです。
修正BSDライセンスに従います。

コンパイラ
==========
 - VisualC++ 2005以降
 - GCC 4.1以降

使用方法
========
jitasm.hを適当なフォルダにコピーしincludeするだけです。
具体的な使用例はsampleフォルダ内のコードを参照してください。
